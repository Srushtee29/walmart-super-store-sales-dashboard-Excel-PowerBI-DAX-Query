# walmart-super-store-sales-dashboard-Excel-PowerBI-DAX-Query
Developed an interactive sales analytics dashboard using Excel, SQL, Power BI, and DAX to analyze sales performance, trends, and KPIs. Implemented time-series forecasting to predict future sales and support data-driven business decisions.
# Walmart Sales Analysis Dashboard 

## Project Overview
Analyzed Walmart sales data to uncover trends and business insights using Power BI and SQL.

## Problem Statement
Walmart wants to understand sales trends, top-performing categories, and regional performance
to make data-driven business decisions.

## Tools Used
- Power BI
- MySQL
- Microsoft Excel

## Key Highlights
- 9,000+ sales records analyzed
- Interactive Power BI dashboard
- Sales trend & category analysis
- forecast Prediction

## Key Insights
- The Walmart sales dashboard reveals that the Consumer segment and West region are the largest contributors to overall revenue. 
- Office Supplies drive the highest sales among categories, while Cash on Delivery remains the most preferred payment method. 
- Sales and profit show a steady upward trend from 2019 to 2020
